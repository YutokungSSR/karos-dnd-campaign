# มหาคัมภีร์แห่งออดมา — D&D Campaign System

ระบบแคมเปญ D&D แบบเต็มสำหรับจักรวาลของ Karos สร้างด้วย Next.js 16 + Supabase + Vercel

## ฟังก์ชันในเวอร์ชันนี้

- สมัครสมาชิกและเข้าสู่ระบบด้วยอีเมล
- DM สร้างแคมเปญและได้รับรหัสเชิญ 8 ตัว
- ผู้เล่นเข้าร่วมแคมเปญด้วยรหัสเชิญ
- สร้างตัวละครได้หลายตัวและผูกกับแคมเปญ
- หน้าต่างสเตตัสแฟนตาซี พร้อมรูปตัวละคร
- HP, MP และ STR / VIT / AGI / INT / DEX / WIS / CHA
- เพิ่ม แก้ไข และลบสกิลกับสถานะผิดปกติ
- คลังไอเทมแบบ MMORPG เริ่มต้น 10 ช่อง พร้อมหมวดอาหาร อาวุธ เครื่องสวมใส่ และไอเทม
- ช่องสวมใส่ 8 ตำแหน่ง: หมวก สร้อย เสื้อ/เกราะ แหวน กางเกง รองเท้า มือซ้าย และมือขวา โดยไอเทมใส่ได้เฉพาะตำแหน่งที่กำหนด
- ผู้เล่นเพิ่มไอเทมพร้อมภาพและจำนวนลงคลังของตัวเอง รวมถึงสวม/ถอดอุปกรณ์ได้เอง
- DM แก้ไขหรือลบไอเทม จัดการอุปกรณ์ และปรับความจุคลังของผู้เล่นได้ทั้งหมด
- DM ปรับ HP ของสมาชิกในหน้าแคมเปญ
- ทอยลูกเต๋า เช่น 1d20, 2d6+3 พร้อมประวัติการทอย
- แชร์ตัวละครแบบสาธารณะด้วยลิงก์ดูอย่างเดียว
- Realtime สำหรับ HP และประวัติการทอย
- Row Level Security ป้องกันผู้ใช้เข้าถึงหรือแก้ข้อมูลที่ไม่มีสิทธิ์
- อัปโหลดภาพผ่าน Supabase Storage
- รองรับคอมพิวเตอร์และโทรศัพท์

## เริ่มใช้งานในเครื่อง

1. สร้างโปรเจกต์ Supabase
2. เปิด SQL Editor และรันไฟล์ `supabase/schema.sql` ทั้งไฟล์
3. คัดลอก `.env.example` เป็น `.env.local`
4. ใส่ Project URL และ anon key จาก Supabase
5. รันคำสั่ง:

```bash
npm install
npm run dev
```

6. เปิด `http://localhost:3000`

อ่านขั้นตอนนำขึ้นออนไลน์แบบละเอียดใน `DEPLOY_TH.md`

## อัปเดตฐานข้อมูลเดิม

ถ้าเว็บไซต์เคยติดตั้ง `schema.sql` มาก่อนแล้ว ให้รัน migration ล่าสุดในโฟลเดอร์ `supabase/migrations` ผ่าน Supabase ก่อน Deploy โค้ดเวอร์ชันใหม่ ไม่ต้องรัน `schema.sql` ทั้งไฟล์ซ้ำบนฐานข้อมูลจริง
